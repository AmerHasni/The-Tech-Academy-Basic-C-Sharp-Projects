# The-Tech-Academy-Basic-C-Sharp-Projects

## Description
This repository contains my projects and exercises completed during a C# and .NET course. It showcases practical applications of core concepts like object-oriented programming, web development with ASP.NET MVC, and database integration using Entity Framework.

## 🚀 Projects
Here is a detailed list of the projects and exercises included in this repository:

| Project Name | Description | Key Skills |
| :--- | :--- | :--- |
| **Car Insurance Quote (Final)** | A comprehensive full-stack application that calculates insurance rates based on user demographics and history. | ASP.NET MVC, Entity Framework, SQL Server |
| **MVC Application** | A web-based project focusing on the Model-View-Controller architectural pattern. | C#, HTML, Web Logic |
| **Polymorphism** | An implementation demonstrating how different classes can be treated as instances of the same interface or base class. | OOP, Interfaces, Inheritance |
| **Operators Overload** | A project demonstrating how to implement custom logic for standard operators within C# classes. | C#, Class Logic |
| **Method Class** | A set of exercises focusing on creating methods, using parameters, and returning data types efficiently. | C#, Logic flow |
| **Branching Assignment** | A project focused on complex conditional logic and control flow to guide user experience. | C#, Logic, Input/Output |

## Features
* **Object-Oriented Programming (OOP):** Encapsulation, inheritance, polymorphism, and interfaces.
* **Web Development:** Building multi-page applications using ASP.NET MVC.
* **Full-Stack Development:** Form handling, CRUD operations, business logic implementation, and SQL Server integration.
* **Real-World Projects:** Includes a car insurance quote application that demonstrates full-stack development.

## Technologies Used
* C#
* .NET Framework
* ASP.NET MVC
* Entity Framework
* SQL Server
